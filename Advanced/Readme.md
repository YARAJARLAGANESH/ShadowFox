# NLP Language Model Analysis (GPT-2 vs BERT)

## 📌 Project Overview
This project explores and analyzes two language models: GPT-2 and BERT. 
The goal is to understand their performance, strengths, and limitations 
in different NLP tasks such as text generation and sentiment analysis.

## ⚙️ Models Used
- GPT-2 (Text Generation)
- BERT (Context Understanding & Classification)

## 🔍 Key Experiments
- Text generation using GPT-2
- Mask prediction using BERT
- Sentiment analysis on IMDb dataset
- Performance comparison (accuracy & latency)
- Failure case analysis

## 📊 Results
- GPT-2 performs better in creative text generation
- BERT performs better in understanding and classification tasks
- Each model has limitations depending on context

## 🧠 Conclusion
The choice of language model depends on the task:
- GPT-2 → generation tasks
- BERT → understanding and classification
